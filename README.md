## 📦 Getting Started

To run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/oleksiinutlis/ecosystems_generation
cd ecosystems_generation
```

### 2️⃣ Run the application

#### 💻 On Windows:
```bash
gradlew.bat run
```

#### 🍎 On Mac/Linux:
```bash
./gradlew run
```
